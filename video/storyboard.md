## Storyboard Panels

### Panel 1: Brand Introduction (00:10 - 00:15)
* **Title Slide**: Clean, Apple-style backdrop featuring a minimalist dark blue symbol representing cognitive memory connections.
* **Vibe**: Elite enterprise software, high trustworthiness.
* **Transition**: Floating nodes connect via pulsing lines, then fade out.

### Panel 2: Live In-Call Dashboard Workbench (00:30 - 00:45)
* **Active View**: Highlights the real-time agent dashboard mockup, including the CSS sound visualizer waveform and live transcription feed.
* **Objection Cards**: Dynamic cards populate on the right column showing objection tags (e.g. pricing, integration) and suggested battlecard script overlays.
* **Transition**: Highlight lines glow, showing instant <1s translation of raw audio.
